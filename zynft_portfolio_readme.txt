## 🧩 ZenChain x ZyNFT Portfolio

### 🚀 About This Project  
This is a sample portfolio built using the **ZyNFT Marketplace** on **ZenChain Testnet**.  
It demonstrates how to create, list, and showcase your NFTs using ZenChain's cutting-edge infrastructure.

---

### 🖼️ Features
- Connect wallet (Metamask/ZenChain wallet)
- Create and mint NFTs
- List NFTs on ZyNFT Marketplace
- View portfolio of owned or created NFTs
- Supports ZenChain Testnet

---

### 🔗 Live Preview  
🌐 [ZyNFT Marketplace](https://zynft.xyz)

---

### 🛠️ Tech Stack
- `React.js` / `Next.js`  
- `ZenChain SDK`  
- `Web3.js`  
- `ZyNFT API`  
- Tailwind CSS

### ⚙️ How to Run Locally
```bash
git clone https://github.com/your-username/zynft-portfolio.git
cd zynft-portfolio
npm install
npm run dev
```
